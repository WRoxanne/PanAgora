# Clinical Trial Data Analysis

## Overview

This project analyzes clinical trial data to identify molecular targets and classify trials by mechanism of action (MOA) and innovation status.

## Files

| File | Description |
|------|-------------|
| `intern_exercise.csv` | Input dataset (184 clinical trials) |
| `trial_analysis_helper.py` | Analysis helper functions |
| `trial_analysis.ipynb` | Interactive analysis notebook |
| `clinical_trial_results.xlsx` | Output results |
